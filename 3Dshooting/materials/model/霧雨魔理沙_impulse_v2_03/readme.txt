◆使用法について
　MikuMikuDance ver8.03にて動作確認しています。
　MMDは樋口優様が開発されたフリーウェアです。
http://www.geocities.jp/higuchuu4/index.htm

　表情モーフについて、「にやり(左)」と「にやり(右)」は
顔の角度限定のモーフです。それぞれ顔が左向き、右向きで
カメラに写っている状態でないと不自然な形になります。

　IKを利用して帽子物理設定のON/OFFを実現しています。
　IK設定から「帽子移動物理ON」「帽子回転物理ON」を
OFFに設定することで移動・回転の物理設定を切る事ができます。

　帽子は表情モーフの「帽子隠し」で小さく、「帽子透明化」で
透明化して隠すことができます。
　しかしこのままでは帽子の剛体が頭上に残っていますので、
「帽子移動用」ボーンで邪魔にならない位置まで移動させて下さい。
